PythonとPolarsを用いたデータ分析と可視化

[出力物タイトル]
教材概要

[出力物内容]
この教材では、Pythonとデータ分析用ライブラリPolarsを用いてオープンデータを分析、可視化する方法を学びます。具体的には、データの整理、集約、フィルタリング、ソートに加え、基本的な統計量の算出とデータの可視化を行います。

[出力物タイトル]
教材テキスト

[出力物内容]
1. データの読み込み
まずは読み込むデータを確認しましょう。PythonとPolarsを使ってデータを読み込みます。

2. データの整理
読み込んだデータを見やすい形に整理します。必要なデータを選択し、不要なデータを削除します。

3. データの集約
集約関数を使ってデータをグルーピングします。これによりデータの特徴を把握します。

4. データのフィルタリングとソート
条件を設定してデータをフィルタリングし、ソートを行います。これによりデータを詳細に分析します。

5. 基本統計量の算出
基本的な統計量を算出します。これによりデータの傾向を確認します。

6. データの可視化
最後に、データを可視化します。これによりデータの全体像を把握します。

[出力物タイトル]
サンプルコード

[出力物内容]
```python
import polars as pl

# データの読み込み
df = pl.read_csv("data.csv")

# データの整理
df = df.select(["column1", "column2"])

# データの集約
df = df.groupby("column1").agg(pl.col("column2").sum())

# データのフィルタリングとソート
df = df.filter(pl.col("column1") > 100).sort("column2")

# 基本統計量の算出
df.describe()

# データの可視化
df.plot()
```
